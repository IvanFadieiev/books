== My Book Application

This is my test application for Sloboda Studio:

* <tt>$ git clone git://github.com/IvanFadieiev/books.git</tt>
* create <tt>/config/database.yml</tt> and <tt>/config/secrets.yml</tt> to your app
* <tt>$ rake db:create</tt>
* <tt>$ rake db:migrate</tt>
* <tt>$ rake db:seed</tt>
* <tt>$ rails s</tt>
